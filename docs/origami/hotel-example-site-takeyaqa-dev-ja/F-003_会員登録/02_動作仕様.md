## Must（できなければならない）

### M-001: フォーム要素の表示

| No | 項目 | 信頼性 |
|----|------|--------|
| M-001-01 | メールアドレス入力フィールドが表示されること | 🟢 |
| M-001-02 | パスワード入力フィールドが表示されること | 🟢 |
| M-001-03 | パスワード（確認）入力フィールドが表示されること | 🟢 |
| M-001-04 | 氏名入力フィールドが表示されること | 🟢 |
| M-001-05 | 会員種別ラジオボタン（プレミアム会員/一般会員）が表示されること | 🟢 |
| M-001-06 | 住所入力フィールドが表示されること | 🟢 |
| M-001-07 | 電話番号入力フィールドが表示されること | 🟢 |
| M-001-08 | 性別ラジオボタン（回答しない/男性/女性/その他）が表示されること | 🟢 |
| M-001-09 | 生年月日入力フィールドが表示されること | 🟢 |
| M-001-10 | 「お知らせを受け取る」チェックボックスが表示されること | 🟢 |
| M-001-11 | 「登録」ボタンが表示されること | 🟢 |

### M-002: 必須入力チェック

| No | 項目 | 信頼性 |
|----|------|--------|
| M-002-01 | メールアドレスが必須入力であること | 🟢 |
| M-002-02 | パスワードが必須入力であること | 🟢 |
| M-002-03 | パスワード（確認）が必須入力であること | 🟢 |
| M-002-04 | 氏名が必須入力であること | 🟢 |

### M-003: バリデーション

| No | 項目 | 信頼性 |
|----|------|--------|
| M-003-01 | パスワードが8文字以上で入力できること | 🟢 |
| M-003-02 | パスワード（確認）がパスワードと一致すること | 🟡 |
| M-003-03 | 電話番号が11桁の数字で入力できること | 🟢 |
| M-003-04 | メールアドレスがメールアドレス形式で入力できること | 🟡 |
| M-003-05 | バリデーションエラー時にエラーメッセージが表示されること | 🟡 |

### M-004: 正常登録

| No | 項目 | 信頼性 |
|----|------|--------|
| M-004-01 | 必須項目のみ入力して会員登録ができること | 🟢 |
| M-004-02 | 全項目を入力して会員登録ができること | 🟢 |
| M-004-03 | プレミアム会員として登録ができること | 🟢 |
| M-004-04 | 一般会員として登録ができること | 🟢 |

### M-005: データ保存

| No | 項目 | 信頼性 |
|----|------|--------|
| M-005-01 | 登録データがLocal Storageに保存されること | 🟢 |
| M-005-02 | 登録したメールアドレスとパスワードでログインできること | 🟢 |

### M-006: 画面表示

| No | 項目 | 信頼性 |
|----|------|--------|
| M-006-01 | ページタイトルが「会員登録 | HOTEL PLANISPHERE - テスト自動化練習サイト」であること | 🟢 |
| M-006-02 | ナビゲーションに「会員登録」リンクが表示されること（未ログイン時） | 🟢 |
| M-006-03 | パスワードフィールドの説明文「8文字以上で入力してください」が表示されること | 🟢 |
| M-006-04 | パスワード（確認）フィールドの説明文「確認のため再度パスワードを入力してください」が表示されること | 🟢 |
| M-006-05 | 電話番号フィールドの入力例「例：01133335555」が表示されること | 🟢 |

### M-007: 初期状態

| No | 項目 | 信頼性 |
|----|------|--------|
| M-007-01 | 会員種別にデフォルト選択値が設定されていること | 🟡 |
| M-007-02 | 性別にデフォルト選択値が設定されていること | 🟡 |
| M-007-03 | 各入力フィールドが空の状態で表示されること | 🟡 |

### M-008: 画面遷移

| No | 項目 | 信頼性 |
|----|------|--------|
| M-008-01 | 登録成功後に適切なページ（マイページまたはトップページ）に遷移すること | 🟡 |
| M-008-02 | ナビゲーションの各リンクから対応するページに遷移できること | 🟢 |

## Never（できてはいけない）

### N-001: 必須項目の空登録

| No | 項目 | 信頼性 |
|----|------|--------|
| N-001-01 | メールアドレスが空の状態で登録できてはいけない | 🟢 |
| N-001-02 | パスワードが空の状態で登録できてはいけない | 🟢 |
| N-001-03 | パスワード（確認）が空の状態で登録できてはいけない | 🟢 |
| N-001-04 | 氏名が空の状態で登録できてはいけない | 🟢 |

### N-002: バリデーション違反での登録

| No | 項目 | 信頼性 |
|----|------|--------|
| N-002-01 | パスワードが8文字未満で登録できてはいけない | 🟢 |
| N-002-02 | パスワード（確認）がパスワードと不一致の状態で登録できてはいけない | 🟡 |
| N-002-03 | 電話番号が11桁の数字以外の値で登録できてはいけない | 🟢 |
| N-002-04 | メールアドレスが不正な形式で登録できてはいけない | 🟡 |

### N-003: データ不整合

| No | 項目 | 信頼性 |
|----|------|--------|
| N-003-01 | 入力した内容と異なるデータがLocal Storageに保存されてはいけない | 🟡 |
| N-003-02 | 既に登録済みのメールアドレスで重複登録できてはいけない | 🟡 |

### N-004: セキュリティ

| No | 項目 | 信頼性 |
|----|------|--------|
| N-004-01 | パスワードが画面上に平文で表示されてはいけない | 🟡 |
| N-004-02 | パスワード（確認）が画面上に平文で表示されてはいけない | 🟡 |
