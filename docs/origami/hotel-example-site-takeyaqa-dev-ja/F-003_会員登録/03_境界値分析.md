## 備考

- 会員登録機能はフォーム入力項目が多いため、境界値分析の対象パラメータも多い
- パスワードの8文字境界（7文字=NG、8文字=OK）が最も明確な境界値
- 電話番号の11桁境界（10桁=NG、11桁=OK、12桁=NG）も明確
- メールアドレス形式チェック、パスワード確認一致チェック、重複登録チェックは仕様書に明示がないため🟡
- 生年月日の有効範囲は仕様書に記載がないため、境界値の期待結果は推測
