# F-002 ログアウト 機能詳細

## 基本情報

| 項目 | 内容 |
|------|------|
| 機能ID | F-002 |
| 機能名 | ログアウト |
| 対象ページ | ナビゲーションメニュー（全ページ共通） |
| 入力ソース | https://hotel-example-site.takeyaqa.dev/ja/ |

---

## 機能概要

ログイン状態のセッションを終了する機能。ナビゲーションメニューの「ログアウト」リンクから実行する。ログアウト後はCookieがクリアされ、ナビゲーションが未ログイン状態に変化する。

---

## 画面構成

### ログアウトリンク

🟢 ログイン後のナビゲーションメニューに「ログアウト」リンクが表示される

### ナビゲーション（ログイン時 = ログアウト実行前）

🟢 以下のメニュー項目が表示される:

| メニュー項目 | リンク先 |
|-------------|---------|
| ホーム | トップページ |
| 宿泊予約 | プラン一覧ページ |
| マイページ | マイページ |
| ログアウト | ログアウト処理 |

### ナビゲーション（ログアウト後）

🟢 ログアウト後は以下のメニュー項目に変化する:

| メニュー項目 | リンク先 |
|-------------|---------|
| ホーム | トップページ |
| 宿泊予約 | プラン一覧ページ |
| 会員登録 | 会員登録ページ |
| ログイン | ログインページ |

---

## ログアウト仕様

### トリガー

🟢 ナビゲーションメニューの「ログアウト」リンクをクリック

### 認証情報のクリア

🟢 ログイン情報はCookieに保存されているため、ログアウト時にCookieがクリアされる

### データ保存方式

🟢 サイト全体のデータ保存方式:

| 保存先 | 説明 |
|--------|------|
| Cookie | ログイン情報の保存 |
| Session Storage | セッションデータの保存 |
| Local Storage | 永続データの保存（会員登録情報等） |

※ サーバーサイドデータベースは使用していない（クライアントサイドのみ）

---

## 画面遷移

### ログアウト後の遷移先

🟡 ログアウト後の遷移先（トップページへ遷移すると推測。仕様書に明示的な記載なし）

---

## 前提条件

🟢 ログイン済み状態であること（ログインしていない場合、ナビゲーションに「ログアウト」リンクは表示されない）
