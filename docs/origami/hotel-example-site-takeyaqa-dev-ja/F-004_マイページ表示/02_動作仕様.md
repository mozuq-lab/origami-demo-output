# F-004 マイページ表示 動作仕様

## 基本情報

| 項目 | 内容 |
|------|------|
| 機能ID | F-004 |
| 機能名 | マイページ表示 |
| 入力ファイル | 01_機能詳細.md |

---

## Must（できなければならない）

### M-001: ページ表示

| No | 項目 | 信頼性 |
|----|------|--------|
| M-001-01 | ページタイトルが「マイページ \| HOTEL PLANISPHERE - テスト自動化練習サイト」であること | 🟢 |
| M-001-02 | ログイン済み状態でマイページが表示されること | 🟢 |
| M-001-03 | ナビゲーションに「ホーム」リンクが表示されること | 🟢 |
| M-001-04 | ナビゲーションに「宿泊予約」リンクが表示されること | 🟢 |
| M-001-05 | ナビゲーションに「マイページ」リンクが表示されること | 🟢 |
| M-001-06 | ナビゲーションに「ログアウト」リンクが表示されること | 🟢 |

### M-002: ユーザー情報の表示（8項目）

| No | 項目 | 信頼性 |
|----|------|--------|
| M-002-01 | ログインユーザーのメールアドレスが表示されること | 🟢 |
| M-002-02 | ログインユーザーの氏名が表示されること | 🟢 |
| M-002-03 | ログインユーザーの会員ランクが表示されること | 🟢 |
| M-002-04 | ログインユーザーの住所が表示されること | 🟢 |
| M-002-05 | ログインユーザーの電話番号が表示されること | 🟢 |
| M-002-06 | ログインユーザーの性別が表示されること | 🟢 |
| M-002-07 | ログインユーザーの生年月日が表示されること | 🟢 |
| M-002-08 | ログインユーザーのお知らせ設定が表示されること | 🟢 |

### M-003: 表示データの正確性

| No | 項目 | 信頼性 |
|----|------|--------|
| M-003-01 | 表示されるメールアドレスがログインユーザーのものと一致すること | 🟢 |
| M-003-02 | 表示される氏名がログインユーザーのものと一致すること | 🟢 |
| M-003-03 | 表示される会員ランクがログインユーザーのものと一致すること（プレミアム会員/一般会員） | 🟢 |
| M-003-04 | 登録した情報がマイページに正確に反映されること | 🟢 |

### M-004: 新規登録ユーザー向け機能

| No | 項目 | 信頼性 |
|----|------|--------|
| M-004-01 | 新規登録ユーザーのマイページに「アイコン設定」リンクが表示されること | 🟢 |
| M-004-02 | 新規登録ユーザーのマイページに「退会する」ボタンが表示されること | 🟢 |
| M-004-03 | 「アイコン設定」リンクをクリックするとアイコン設定ページ（icon.html）に遷移すること | 🟢 |

### M-005: 任意項目の表示

| No | 項目 | 信頼性 |
|----|------|--------|
| M-005-01 | 住所が登録されている場合、登録値が表示されること | 🟡 |
| M-005-02 | 電話番号が登録されている場合、登録値が表示されること | 🟡 |
| M-005-03 | 性別が登録されている場合、登録値が表示されること | 🟡 |
| M-005-04 | 生年月日が登録されている場合、登録値が表示されること | 🟡 |
| M-005-05 | 任意項目が未入力の場合、適切な表示（空欄・ハイフン等）がなされること | 🟡 |

### M-006: ナビゲーション遷移

| No | 項目 | 信頼性 |
|----|------|--------|
| M-006-01 | 「ホーム」リンクからトップページに遷移できること | 🟢 |
| M-006-02 | 「宿泊予約」リンクからプラン一覧ページに遷移できること | 🟢 |
| M-006-03 | 「マイページ」リンクからマイページに遷移できること | 🟢 |
| M-006-04 | 「ログアウト」リンクからログアウトできること | 🟢 |

---

## Never（できてはいけない）

### N-001: 未ログイン時のアクセス制限

| No | 項目 | 信頼性 |
|----|------|--------|
| N-001-01 | 未ログイン状態でマイページの情報が表示されてはいけない | 🟡 |

### N-002: 表示データの不整合

| No | 項目 | 信頼性 |
|----|------|--------|
| N-002-01 | 他のユーザーの情報がマイページに表示されてはいけない | 🟢 |
| N-002-02 | 登録した情報と異なるデータがマイページに表示されてはいけない | 🟢 |

### N-003: 事前登録ユーザーへの機能制限

| No | 項目 | 信頼性 |
|----|------|--------|
| N-003-01 | 事前登録テストユーザーのマイページに「アイコン設定」リンクが表示されてはいけない | 🟢 |
| N-003-02 | 事前登録テストユーザーのマイページに「退会する」ボタンが表示されてはいけない | 🟢 |

### N-004: セキュリティ

| No | 項目 | 信頼性 |
|----|------|--------|
| N-004-01 | 他のユーザーのLocal Storageデータを読み取って表示してはいけない | 🟡 |
