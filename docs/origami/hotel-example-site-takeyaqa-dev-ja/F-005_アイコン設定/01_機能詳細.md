# F-005 アイコン設定 機能詳細

## 基本情報

| 項目 | 内容 |
|------|------|
| 機能ID | F-005 |
| 機能名 | アイコン設定 |
| 対象ページ | icon.html |
| 入力ソース | https://hotel-example-site.takeyaqa.dev/ja/icon.html |

---

## 機能概要

新規登録ユーザーがプロフィールアイコン画像をアップロードし、拡大・縮小・枠線色の設定を行い保存する機能。マイページの「アイコン設定」リンクから遷移し、設定した画像はプロフィールアイコンとして使用される。

---

## 利用条件

🟢 新規登録ユーザーのみ利用可能（事前登録テストユーザーには提供されない）
🟢 マイページ（mypage.html）の「アイコン設定」リンクからアクセス
🟡 ログイン済み状態であること（ログインなしの直接アクセス時の挙動は仕様書に明示なし）

---

## UI要素

### ファイルアップロード

🟢 アイコン画像ファイルアップロードフィールドが存在

| 項目 | 内容 |
|------|------|
| ラベル | アイコン画像 |
| 説明文 | 🟢 「10KB以下の画像ファイルを選択してください」 |
| ファイルサイズ制限 | 🟢 10KB以下 |
| 対応ファイル形式 | 🟡 画像ファイル（PNG・JPEG・GIF等と推測、input type="file" accept="image/*"と推測） |

### 拡大・縮小スライダー

🟢 拡大・縮小機能のスライダーが存在

| 項目 | 内容 |
|------|------|
| ラベル | 拡大・縮小 |
| 入力形式 | 🟡 スライダー（range input と推測） |
| 最小値 | 🟡 不明（仕様書に記載なし） |
| 最大値 | 🟡 不明（仕様書に記載なし） |
| 初期値 | 🟡 不明（仕様書に記載なし） |
| 動作 | 🟡 プレビュー画像をリアルタイムで拡大・縮小すると推測 |

### 枠線の色設定

🟢 枠線の色設定UIが存在

| 項目 | 内容 |
|------|------|
| ラベル | 枠線の色 |
| 入力形式 | 🟡 カラーピッカーまたは色選択リスト（選択肢の詳細は仕様書に記載なし） |
| 選択肢 | 🟡 複数の色オプション（詳細不明） |
| 動作 | 🟡 プレビュー画像の枠線色をリアルタイムで変更すると推測 |

### プレビューエリア

🟡 アップロードした画像のプレビューが表示されると推測（拡大・縮小・枠線色の変更が反映される）

### 確定ボタン

🟢 「確定」ボタンが存在

| 項目 | 内容 |
|------|------|
| ラベル | 確定 |
| 動作 | 🟡 設定内容をLocal Storage等に保存し、マイページへ遷移すると推測 |

---

## 動作仕様

### ファイルサイズバリデーション

🟢 10KB以下のファイルのみ受け付ける（説明文から明示的に確認済み）
🟡 10KBを超えるファイルを選択した場合のエラー表示方式（仕様書に記載なし）

### ファイル形式バリデーション

🟡 画像ファイル以外を選択した場合のエラー表示方式（仕様書に記載なし）

### 保存処理

🟡 「確定」ボタンクリック後の保存先（Local Storageと推測）
🟡 「確定」ボタンクリック後の遷移先（マイページへ遷移と推測）

---

## 前提条件

🟢 新規登録ユーザーとしてログイン済みであること
🟢 マイページの「アイコン設定」リンクからの遷移を経由する通常フロー
