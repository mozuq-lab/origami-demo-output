# F-005 アイコン設定 動作仕様

## 基本情報

| 項目 | 内容 |
|------|------|
| 機能ID | F-005 |
| 機能名 | アイコン設定 |
| 入力ファイル | 01_機能詳細.md |

---

## Must（できなければならない）

### M-001: ページ表示

| No | 項目 | 信頼性 |
|----|------|--------|
| M-001-01 | ページタイトルが「アイコン設定 \| HOTEL PLANISPHERE - テスト自動化練習サイト」であること | 🟢 |
| M-001-02 | 新規登録ユーザーがアイコン設定ページにアクセスできること | 🟢 |
| M-001-03 | ナビゲーションに「ホーム」リンクが表示されること | 🟢 |
| M-001-04 | ナビゲーションに「宿泊予約」リンクが表示されること | 🟢 |
| M-001-05 | ナビゲーションに「マイページ」リンクが表示されること | 🟢 |
| M-001-06 | ナビゲーションに「ログアウト」リンクが表示されること | 🟢 |

### M-002: UI要素の表示

| No | 項目 | 信頼性 |
|----|------|--------|
| M-002-01 | 「アイコン画像」ファイルアップロードフィールドが表示されること | 🟢 |
| M-002-02 | 「10KB以下の画像ファイルを選択してください」という説明文が表示されること | 🟢 |
| M-002-03 | 「拡大・縮小」スライダーが表示されること | 🟢 |
| M-002-04 | 「枠線の色」設定UIが表示されること | 🟢 |
| M-002-05 | 「確定」ボタンが表示されること | 🟢 |
| M-002-06 | アップロードした画像のプレビューが表示されること | 🟡 |

### M-003: ファイルアップロード（正常系）

| No | 項目 | 信頼性 |
|----|------|--------|
| M-003-01 | 10KB以下の画像ファイルをアップロードできること | 🟢 |
| M-003-02 | アップロードした画像がプレビューエリアに表示されること | 🟡 |

### M-004: 拡大・縮小機能

| No | 項目 | 信頼性 |
|----|------|--------|
| M-004-01 | スライダーを操作してプレビュー画像を拡大できること | 🟡 |
| M-004-02 | スライダーを操作してプレビュー画像を縮小できること | 🟡 |
| M-004-03 | スライダー操作がリアルタイムでプレビューに反映されること | 🟡 |

### M-005: 枠線の色設定

| No | 項目 | 信頼性 |
|----|------|--------|
| M-005-01 | 枠線の色を選択できること | 🟢 |
| M-005-02 | 選択した枠線の色がプレビューに反映されること | 🟡 |

### M-006: 確定処理

| No | 項目 | 信頼性 |
|----|------|--------|
| M-006-01 | 「確定」ボタンをクリックするとアイコン設定が保存されること | 🟡 |
| M-006-02 | 「確定」ボタンクリック後に適切なページ（マイページ等）に遷移すること | 🟡 |

### M-007: ナビゲーション遷移

| No | 項目 | 信頼性 |
|----|------|--------|
| M-007-01 | 「マイページ」リンクからマイページに遷移できること | 🟢 |
| M-007-02 | 「ホーム」リンクからトップページに遷移できること | 🟢 |

---

## Never（できてはいけない）

### N-001: ファイルサイズ制限違反

| No | 項目 | 信頼性 |
|----|------|--------|
| N-001-01 | 10KBを超えるファイルがアイコンとして設定されてはいけない | 🟢 |
| N-001-02 | 10KBを超えるファイルをアップロードしようとした際にエラーなく受理されてはいけない | 🟢 |

### N-002: ファイル形式制限

| No | 項目 | 信頼性 |
|----|------|--------|
| N-002-01 | 画像ファイル以外（テキスト、PDF等）がアイコンとして設定されてはいけない | 🟡 |

### N-003: アクセス制限

| No | 項目 | 信頼性 |
|----|------|--------|
| N-003-01 | 事前登録テストユーザーがアイコン設定ページを利用できてはいけない（マイページからのリンクが非表示） | 🟢 |
| N-003-02 | 未ログイン状態でアイコン設定ページが利用できてはいけない | 🟡 |
