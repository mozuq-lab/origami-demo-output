# F-001 ログイン 機能詳細

## 基本情報

| 項目 | 内容 |
|------|------|
| 機能ID | F-001 |
| 機能名 | ログイン |
| 対象ページ | login.html |
| 入力ソース | https://hotel-example-site.takeyaqa.dev/ja/ |

---

## 機能概要

メールアドレスとパスワードによるユーザー認証機能。認証情報はCookieに保存され、ログイン状態によりナビゲーションメニューの表示内容が変化する。

---

## 画面構成

### ページタイトル

🟢 「ログイン | HOTEL PLANISPHERE - テスト自動化練習サイト」

### ナビゲーション（未ログイン時）

🟢 以下のメニュー項目が表示される:

| メニュー項目 | リンク先 |
|-------------|---------|
| ホーム | トップページ |
| 宿泊予約 | プラン一覧ページ |
| 会員登録 | 会員登録ページ |
| ログイン | ログインページ（現在ページ） |

### ナビゲーション（ログイン後）

🟢 ログイン後は以下のメニュー項目に変化する:

| メニュー項目 | リンク先 |
|-------------|---------|
| ホーム | トップページ |
| 宿泊予約 | プラン一覧ページ |
| マイページ | マイページ |
| ログアウト | ログアウト処理 |

### フォーム要素

🟢 以下の入力要素で構成される:

| 要素 | 種別 | 説明 |
|------|------|------|
| メールアドレス | テキスト入力 | メールアドレスの入力フィールド |
| パスワード | パスワード入力 | パスワードの入力フィールド |
| ログインボタン | ボタン | 認証処理を実行するボタン |

---

## 認証仕様

### 認証方式

🟢 Cookie ベースの認証

- 認証情報はCookieに保存される
- ログイン状態の維持にCookieを使用

### 登録済みテストユーザー

🟢 以下の4名のテストユーザーが事前登録されている:

| No. | メールアドレス | パスワード | 会員ランク |
|-----|--------------|-----------|-----------|
| 1 | ichiro@example.com | password | プレミアム会員 |
| 2 | sakura@example.com | pass1234 | 一般会員 |
| 3 | jun@example.com | pa55w0rd! | プレミアム会員 |
| 4 | yoshiki@example.com | pass-pass | 一般会員 |

### 会員ランク

🟢 2種類の会員ランクが存在する:

| ランク | 説明 |
|--------|------|
| プレミアム会員 | 上位会員。宿泊プラン一覧で表示されるプランが増える |
| 一般会員 | 通常会員 |

## 画面遷移

### ログイン成功時

🟡 ログイン成功後の遷移先（トップページへ遷移すると推測。仕様書に明示的な記載なし）

### ログイン失敗時

🟡 ログイン失敗時の挙動（エラーメッセージ表示と推測。具体的なメッセージ内容は仕様書に明示的な記載なし）

## バリデーション

🟡 入力バリデーションの詳細（仕様書に明示的な記載なし。以下は一般的なログインフォームの挙動から推測）:

|------|----------------------|
